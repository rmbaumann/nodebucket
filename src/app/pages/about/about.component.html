Nodebucket was a project created during a capstone class at Bellevue University in 2019. Nodebucket is a simple task management system that provides the means for individuals to track and complete tasks.

Nodebucket provides the means to create, track, delete and complete everday tasks. The system was built using a MEAN stack, using MongoDB, Express.js, Angular and Node. We combined a list of languages and skills to complete the project and also simulated a working enviornment, similar to what's actively found in the web development world. Professor Krasso created weekly sprints and used working meetings to create a world similar to what we will actively see once we graduate.

To test nodebucket, Reva called in the Justice League and the likes of DC Comics to create a simple login system and example tasks for the League. These included tasks like:

  • Dryclean Superman's cape.
  • Assign employee evaluations to Batman.
  • Clean and service Justice League Jet.
  • Hang up Wonder Woman's lasso.
